# **App Name**: TalkTree Admin

## Core Features:

- Admin Menu & Submenu Management: Admins can create, edit, update, and delete menu and submenu entries, defining their display names and hierarchical structure.
- WYSIWYG Content Editor: Admins can compose and format rich text content directly within menu or submenu entries using an integrated editor.
- Dynamic Conversational Navigation: Users can navigate through menus and submenus via a chatbot-like interface, selecting options to drill down or view content.
- Nested Content Display: Display either the formatted content of a selected menu item or a list of its child submenus, supporting unlimited nesting.
- Intuitive Navigation Controls: Provides 'Back to Main Menu' and 'Back to Previous Menu' options to ensure easy and contextual user navigation.
- AI Multilingual Content Tool: An AI tool to assist in generating content translations for different languages, available to users for selection.
- User Language Selector: Users can select their preferred language from available options, switching the displayed menu structure and content.

## Style Guidelines:

- Primary color: A clear, friendly blue (#31A3E2) for trust and communication, enhancing the conversational feel.
- Background color: A subtle, cool-toned off-white (#F1F5F8) to ensure high readability and a clean interface.
- Accent color: A vibrant yet thoughtful violet (#564AED) to highlight interactive elements and call-to-action buttons, drawing attention without being overwhelming.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for a text-heavy, chatbot-like interface.
- Use minimalist, outline-style icons for navigation controls (e.g., arrows for 'back', a home icon for 'main menu') and interactive elements, maintaining visual clarity.
- A clean, mobile-first layout with generous white space. Content areas should mimic chat bubbles or concise message blocks for an intuitive conversational flow, especially on the user view.
- Incorporate smooth, subtle transitions for menu expansions, content loading, and button interactions to provide a responsive and fluid user experience.
